# Control de Gastos - Diego 📊

Una aplicación de escritorio intuitiva desarrollada en **Python** diseñada para ayudarte a gestionar tus gastos personales de forma organizada y eficiente, con la capacidad de exportar tus datos directamente a **Microsoft Excel** para un análisis más profundo.

## 🚀 Características Principales

*   **Registro de Gastos:** Añade tus transacciones diarias detallando el monto, la categoría y la fecha.
*   **Interfaz de Escritorio:** Navegación limpia y accesible directamente desde tu computadora.
*   **Exportación a Excel:** Genera reportes en formato `.xlsx` con un solo clic para mantener un respaldo o analizar tus finanzas.
*   **Uso Local:** Tus datos se guardan de forma privada y segura en tu propio equipo.

## 🛠️ Requisitos Previos

Antes de ejecutar la aplicación, asegúrate de tener instalado lo siguiente en tu sistema:

*   [Python 3.x](https://python.org)
*   Librerías necesarias (puedes instalarlas mediante la terminal):
    ```bash
    pip install pandas openpyxl
    ```
    *(Nota: Reemplaza o añade aquí las librerías gráficas o de interfaz que utilices en tu archivo `gastos.py`, por ejemplo `tkinter` o `customtkinter` si corresponde).*

## 💻 Instalación y Uso

Sigue estos sencillos pasos para poner en marcha la aplicación en tu computadora de forma local:

1.  **Clona este repositorio** o descarga los archivos directamente:
    ```bash
    git clone https://github.com
    ```
2.  **Navega al directorio** del proyecto:
    ```bash
    cd control-de-gastos-Diego
    ```
3.  **Ejecuta la aplicación:**
    ```bash
    python gastos.py
    ```

## 📂 Estructura del Proyecto

*   `gastos.py`: Archivo principal que contiene el código fuente de la lógica y la interfaz de la aplicación.
*   `README.md`: Archivo de documentación con las instrucciones de uso (este archivo).
